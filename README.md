# Demonstration-and-Control-Platform
A platform for running hands on the benchtop with full functionality. Designed for both the mobile control stick and benchtop testing platform.

## Demo Stick Software Components
- Demo Stick Manager (DSMananger)
- LCD Display (DSDisplay)
- VBat and VTemp Monitor (DSADC)
- Button Handler (DSButtons)
- Artus Lite API (demo_stick_feat_slave_id branch of Sarcomere Dynamics Resources)
